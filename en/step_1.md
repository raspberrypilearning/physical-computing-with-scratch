### Hardware

- A Raspberry Pi 4
- An LED
- A 47 - 330 ohm resistor
- A tactile button
- A buzzer
- A PIR motion sensor
- Male to Female jumper leads
- Female to Female jumper leads

### Software

Scratch 3 - if not already installed, open a terminal and type:

```bash
sudo apt install scratch3
```
